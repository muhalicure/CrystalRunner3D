# Crystal Runner 3D

Crystal Runner 3D is a simple 3D collectible game developed with Unity 6 and C# for the Game Design and Programming course.

## Gameplay

The player controls a character and tries to collect 5 crystals as quickly as possible.  
After each crystal is collected, it respawns at a random position on the platform.

## Features

- 3D player movement
- Crystal collection system
- Random crystal respawn
- Countdown before gameplay
- Timer system
- Player name input
- Best ranks system
- Persistent score saving with PlayerPrefs
- Start menu and finish screen

## Technologies Used

- Unity 6
- C#
- TextMesh Pro
- Unity UI
- Rigidbody and Collider system
- Blender
